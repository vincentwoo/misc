Pitfalls

  • Chrome/FF only
  • Browser security doesn't let you use the File API when running off of disk, need to run a webserver to echo the files back to you
  • I think my saddle-point resolution is off, would need more time to tinker
  • In fact, my next step would probably be to scrap marching-squares entirely and go for some line-division or bezier interpolation for a better-looking map
